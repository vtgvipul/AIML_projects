    user_ratings = pivot_data.loc[userID].sort_values(ascending = False)
    user_Preds = pred_dataSet.loc[userID].sort_values(ascending = False)
    user_recom_table = pd.concat([user_ratings,user_Preds], axis = 1)
    user_recom_table.index.name = 'Recommended Products'
    user_recom_table.columns = ['Actual_ratings', 'Predicted_ratings']
    user_recom_table = user_recom_table.loc[user_recom_table.Actual_ratings == 0]
    user_recom_table = user_recom_table.sort_values('Predicted_ratings', ascending = False)
    print("Following are the "+ str(n_recommendations) + ' for the user ' + str(userID) + ' :')
    return user_recom_table.head(n_recommendations)

Thus, from the above activity of building the popularity and collaborative based recommendation system, we can conclude and summarise the following:

  1. Popularity based system is generalized and non personal. It will recommend the top N and popular product/ items in the database. No user preferences are taken into consideration.

    1.a. Data sparsity or density won't have much affect on these systems.

  1. Collaborative filtering/ recommenders are user based systems that consider the user preferences by guaging the user's previous ratings and behaviour. The matrix based evaluation helps to predict the rating this user may give to a certain item, to which the user has never rated and on this basis, top N can be recommended to the user.

    2.a. Data density affects the recommendation. More sparse the data - vertically or horizantally will affect. If there are too many products and users, which are sparsely rated, such data will be not effective to predict ratings. For example for a single user, if we try to predict the ratings for Nth item out of a few 10 thousands of products, provided the user has only rated a handful of products.

    2.b Collaborative filtering is helpful in cases where many users have equally rated throughout the oriduct spectrum.

  1. Hybrid recommendations are more efficient where collaborative and content based systems can help to achieve a near actual prediction and recommendation in cases of sparse data.
